# 🎨 Saraha Frontend

The sleek, glassmorphism-inspired web client for Saraha, built with **React**, **Vite**, and **Nx**.

## 🛠 Features
- **Glassmorphism**: Modern, translucent UI components.
- **Dynamic Routing**: Powered by React Router 7.
- **State Management**: Clean hooks and services for auth and messaging.
- **Responsive**: Fully optimized for mobile and desktop.

## 🚀 Getting Started

### Prerequisites
Ensure the backend is running for the app to function correctly.

### Commands
From the project root:

| Command | Description |
| :--- | :--- |
| `nx serve frontend` | Start development server |
| `nx build frontend` | Build for production |
| `nx test frontend` | Run unit tests |
| `nx lint frontend` | Run linter |

## 📁 Architecture
Components are organized into logical features in `src/app/features/` to maintain scalability.
